# Captain's Patch
*Wondrous Item, Rare*

This finely crafted compass is said to be the only navigation tool a traveler will ever need. While attuned to this magical compass you gain the following benefits.

***Location Library.*** The compass will upon command memorise any location you’ve visited. By speaking the name of a memorized location, the compass needle points in that direction until you arrive. You can command the compass to change the active location or return to normal orientation at any time.

***Tracking Needle.*** During a Long or Short Rest you can choose up to four friendly creatures to track with the compass. As long as the tracked creatures are on the same plane as you, the compass will point the way to them upon command.

***Mapping Tools.*** You can cast Locate Object or Locate Creature without expending a spell slot once, this ability recharges daily at Dawn. When casting these spells using the compass.
- You can concentrate for up to 12 hours.
- The spell range is unlimited, as long as the target is on the same plane as you.
